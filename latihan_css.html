Latihan CSS

Ini adalah contoh penggunaan CSS.

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk menyusun struktur dan konten halaman web, termasuk elemen seperti heading, paragraf, dan tautan.

CSS (Cascading Style Sheets) digunakan untuk menentukan gaya dan tata letak elemen HTML, seperti mengubah warna, font, dan jarak, agar halaman web tampil lebih menarik dan sesuai desain.

HTML memberikan kerangka dasar untuk halaman web, sementara CSS mempercantik tampilan dengan berbagai penyesuaian visual agar halaman web menjadi lebih profesional dan responsif.
